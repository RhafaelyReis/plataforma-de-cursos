# Plataforma de Cursos

Projeto desenvolvido para a disciplina de Scrum e LGPD.

## Objetivo

Desenvolver uma plataforma de cursos utilizando metodologias ágeis e aplicando conceitos da LGPD (Lei Geral de Proteção de Dados).

## Funcionalidades

- Cadastro de usuário
- Login
- Listagem de cursos
- Inscrição em cursos
- Acompanhamento de progresso
- Edição de dados pessoais
- Consentimento para marketing
- Revogação de consentimento
- Exclusão de conta
- Logs de autenticação

## Tecnologias Utilizadas

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- Thymeleaf
- MySQL
- Bootstrap

## Organização do Projeto

O projeto será desenvolvido em 3 sprints utilizando Jira para gerenciamento das tarefas.
