# plataforma-de-cursos
A plataforma de cursos é um sistema web desenvolvido para gerenciamento de cursos online, permitindo cadastro e autenticação de usuários, inscrição em cursos e acompanhamento de progresso. O projeto também aplica conceitos da LGPD, incluindo consentimento para marketing, gerenciamento de dados pessoais e armazenamento seguro de senhas.
